### MACアドレスとは・ARPテーブルとは

ゲートウェイのMACは42:01:0a:92:00:01であってる。

- ip link show
- ip neigh show

#### ARPテーブルとは
ネットワーク通信では、2種類のアドレスが混在している

- ipアドレス(L3)  : 10.146.0.1 -> 論理的な住所
- MACアドレス(L2) : 42:01:0a:92:00:01 -> 物理的な機器の識別子

イーサネットでパケットを送るにはMACアドレスが必要。ですが、人間が知っているのはIPアドレスだけです。
そこで、「IPアドレス-> MACアドレスの変換」が必要になる。これがARPテーブル

IPアドレス       MACアドレス
10.146.10.1  ->  42:01:0a:92:00:01
10.146.0.5  →  xx:xx:xx:xx:xx:xx

- どうやってARPテーブルを埋めるのか
「10.146.0.1 の MAC を持つ人、返事して！」とブロードキャスト送信（ARP Request）
10.146.0.1 が「私の MAC は 42:01:0a:92:00:01 です」と返信（ARP Reply）
